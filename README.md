Твикер для сайта [eblo.id](https://eblo.id)

Пока что только для браузеров на движке хромиум

В твикере есть:
- *бесконечная лента*
- *блюр медиа в ленте* (не закончено) 
- *сворачивание коментариев*
- *запоминание громкости*
- *заппоминание поста*
- *изменение сетки постов на главной на 3-4-5 колонки*

Скачайте и распакуйте, потом зайдите в расширения, включите режим разработчика и нажмите загрузить распакованное расширение
выберите папку в которой лежит файл manifest.json

И по идеи если вы перезагрузите сайт eblo.id (если вы находились на нем до включения расширения)
то в меню сайта над настройками будет кнопочка с менюшкой твикера

Все идеи и помощь принимаются ;)
Будте добры, поставьте звездочку
